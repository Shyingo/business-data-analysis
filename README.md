## 📌 项目概述

在数字化浪潮背景下，数据已成为企业核心战略资产。本项目构建一个完整的商业数据分析示范体系，通过模拟真实的商业环境，将散乱的交易数据转化为系统、深入的战略洞察，为企业决策提供科学、量化的支持。

**四大功能模块：**
- **数据生成** — 创建包含客户信息、产品信息和交易记录的完整商业数据集
- **数据探索** — 数据质量检查、缺失值验证、关键业务指标计算
- **可视化分析** — 21种可视化图表，多维度交叉分析
- **洞察生成** — 提炼关键业务发现，生成数据驱动的决策建议

## 📂 项目结构

```
business-data-analysis/
├── generate_data.py      # 模拟数据集生成（客户+产品+交易）
├── explore_data.py       # 数据探索与清洗（质量检查+分布分析+趋势探索）
├── visualize.py          # 可视化分析（21张图表，含RFM/热力图/季节性分析）
├── main.py               # 一键运行入口
├── requirements.txt      # 依赖清单
├── .gitignore
├── data/                 # 原始数据（运行后生成）
│   ├── customers.csv     # 客户表（收入/区域/细分）
│   ├── products.csv      # 产品表（含成本价和利润率）
│   └── transactions.csv  # 交易表（含利润）
└── output/               # 清洗后数据 & 图表（运行后生成）
    ├── customers_clean.csv
    ├── products_clean.csv
    ├── transactions_clean.csv
    ├── rfm_scores.csv
    └── charts/ (21张图表)
```

## 🚀 快速开始

### 1️⃣ 安装依赖

```bash
pip install pandas numpy matplotlib faker
```

### 2️⃣ 一键运行

```bash
python main.py
```

### 3️⃣ 单独执行各模块

```bash
python generate_data.py    # 只生成数据
python explore_data.py     # 只做数据探索与清洗
python visualize.py        # 只生成可视化图表
```

## 🔧 技术要点

- **数据生成**：Faker库生成真实感中文数据；对数正态分布模拟收入；成本=价格×40%-80%
- **数据清洗**：缺失值处理、异常值过滤、类型转换、描述性统计
- **分布分析**：6个核心维度的分布特征
- **RFM模型**：Recency/Frequency/Monetary客户价值分层
- **相关性分析**：热力图展示变量间关系
- **季节性分析**：按月聚合识别周期模式
- **可视化**：21张图表，含双Y轴、子图、饼图、箱线图、散点图等
